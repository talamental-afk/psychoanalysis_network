# Psychoanalysis Network

一个交互式的精神分析概念网络图，旨在帮助用户理解复杂的精神分析理论体系及其概念间的推导关系。

## ✨ 功能特性

- **交互式网络图**：可视化展示精神分析概念及其相互联系。
- **推导链视图**：清晰呈现每个概念的推导来源和路径。
- **移动端适配**：支持触摸交互，响应式布局，在手机和平板上提供流畅体验。
- **Zustand 状态管理**：高效管理应用状态，提升性能和可维护性。
- **模块化架构**：代码结构清晰，易于扩展和维护。

## 🛠️ 本地开发

1.  **克隆仓库**：
    ```bash
    git clone https://github.com/talamental-afk/psychoanalysis_network.git
    cd psychoanalysis_network
    ```

2.  **安装依赖**：
    ```bash
    pnpm install
    ```

3.  **启动开发服务器**：
    ```bash
    pnpm dev
    ```

4.  **访问应用**：
    在浏览器中打开 `http://localhost:5173` (或命令行中显示的地址)。
